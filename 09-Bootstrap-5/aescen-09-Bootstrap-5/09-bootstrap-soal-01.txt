1. Apa itu CSS Framework?
   - CSS Framework adalah kumpulan CSS style yang dipakai untuk membuat tema atau tampilan
     web lebih mudah, praktis, cepat dan sesuai standar desain pada web
   - Terdapat pula CSS framework yang memiliki style atau fitur lebih interaktif
     karena didalamnya juga menggunakan javascript

2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
   - Tidak, ada banyak CSS framework lainnya seperti Bulma, Foundation, Tailwind dll

3. Sebutkan beberapa CSS Framework selain Bootstrap
   - Bulma
   - Foundation
   - Tailwind

4. Bagaimana tata cara penggunaan Bootstrap?
   - Masukkan CSS Bootstrap pada tag head
       <link
         crossorigin="anonymous"
         href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/css/bootstrap.min.css"
         rel="stylesheet"
         integrity="sha384-gH2yIJqKdNHPEq0n4Mqa/HGKIhSkIHeL5AyhkYV8i59U5AR6csBvApHHNl/vI1Bx"
       />
   - Masukkan JS Bootstrap sebelum closing tag body, atau tambahkan atribut defer bila ditaruh di dalam tag head
       <script
         crossorigin="anonymous"
         src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/js/bootstrap.bundle.min.js"
         integrity="sha384-A3rJD856KowSb7dwlZdYEkO39Gagi7vIsF0jrRAoQmDKKtQBHUuLZ9AsSv4jD4Xa"
       ></script>
